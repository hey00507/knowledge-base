## 📝 주요 회의 내용

### 1) 화면 기획 및 일정

- 1차 오픈 목표: 10월 15일
    - 9월 3일 전달된 디자인 시안(PDF) 기준으로 진행
    - 회원 데이터는 임시로 DB 직접 주입 방식
- 2차 범위(B, C안):
    - 이글로벌시스템 정리 → 제타 전달 → 제타 시안 작업 → 고객 확인
    - B안 특이사항: 데이터 이동이 복잡하여 페이지 구성 난이도 ↑
        - 송원기 전임 협조 요청 건으로 김종구, 신은별 별도 미팅 예정

### 2) 고객 테스트 대응

- 긴급: 9월 11일까지 고객 테스트용 페이지 준비
- 개발 가능한 범위 우선 선별하여 제타에서 전달
- API 개발에 필요한 DB 명세서 부족
- 조치: 이글로벌시스템에서 9월 4일까지 DB 명세 정리 후 전달
